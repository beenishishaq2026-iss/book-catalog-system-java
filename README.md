## Book Catalog System
        A web-based Book Catalog System developed as an academic project using Java Servlets and JDBC. The system allows users to manage, search, and organize books through a structured web interface.
##  Features -
         Add, update, delete books - View book details -cart management Database connectivity using JDBC - MVC-based architecture using Servlets ---
## Technology Stack -
        Java (Servlets & JSP) - JDBC - Apache Tomcat - MySQL - HTML, CSS, Bootstrap ---
## How to Run the Project 
1. Install *Apache Tomcat* 
2. Import project into *Eclipse / IntelliJ*
3. Configure database connection
4. Run on Tomcat server 
5. Access via browser ---
##  Academic Context
           This project was developed as part of a university coursework to understand Java web development and database integration.
